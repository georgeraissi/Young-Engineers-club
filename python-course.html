Part 3: Operators and Expressions

Arithmetic Operators
  • + ‚Üí addition
  • - ‚Üí subtraction
  • * ‚Üí multiplication
  • / ‚Üí division (always float)
  • // ‚Üí floor division (cuts off decimals)
  • % ‚Üí remainder (modulo)
  • ** ‚Üí exponent
print(7 // 2)   # 3
print(7 % 2)    # 1
print(2 ** 3)   # 8
Comparison Operators
  • == ‚Üí equal to
  • != ‚Üí not equal to
  • <, >, <=, >=
print(5 == 5)   # True
print(5 != 5)   # False
print(3 < 10)   # True
Boolean Operators
  • and ‚Üí both must be True
  • or ‚Üí at least one must be True
  • not ‚Üí flips True/False
print(5 > 2 and 5 < 10)  # True
print(5 > 2 or 5 < 0)    # True
print(not 5 > 2)         # False
